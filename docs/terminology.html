Units¶

The methods in MHKiT use the MKS (meters-kilograms-seconds) system, and assume data is stored in SI units, for example:

  • Acceleration = \(m/s^2\)

  • Current = \(A\)

  • Distance = \(m\)

  • Energy = \(J\)

  • Frequency = \(Hz\)

  • Mass = \(kg\)

  • Power = \(W\)

  • Pressure = \(Pa\)

  • Time = \(s\)

  • Velocity = \(m/s\)

  • Voltage = \(V\)

  • Volume = \(m^3\)
